## Início Rápido

1. Compre o plugin em [steveplugins.com](https://steveplugins.com)
2. Coloque o `.jar` na pasta `plugins/` do servidor
3. Reinicie o servidor
4. Configure a chave de licença no `config.yml`
5. Pronto — consulte a documentação do plugin para detalhes
